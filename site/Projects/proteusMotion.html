I am currently interning at Proteus Motion working on the 3D visuals for their latest device, the Proteus System. I am refactoring their Three.js code into Babylon.js so that they can offload some processes into a separate thread using Babylon's Offscreen Canvas. This is desirable because it will help their biofeedback system run faster. I am using the data they have given me to construct 3D spaces and objects that will help their users analyze their exercises.
